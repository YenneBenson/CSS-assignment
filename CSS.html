The Cascading Style Sheets (CSS) is used to give a web page a more appealing look, and to put in place different user experience concepts.

In the CSS box model, there are different concepts, some of which are padding, margin and borders. Padding is the space around an element, inside a defined border. Margin is the space around the border and element. Border covers the element and the padding.
